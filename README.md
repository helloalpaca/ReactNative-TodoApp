# ReactNative-TodoApp

React Native 개발 환경 설정은 다음 [문서](https://reactnative.dev/docs/environment-setup)를 참고하세요.

#### Package Install
```
npm install
```

#### RUN iOS
```
cd pod
pod install
cd ..
```

```
npm run ios
```

#### RUN Android
```
npm run android
```
